# Advanced AI Air Quality Monitoring System

## Project Description
This project is an AI-powered air quality monitoring system.
It predicts future AQI values using machine learning,
analyzes health risks, sends alerts, and shows air quality on a map.
The system helps people understand pollution levels easily.

## Features
1. AQI prediction using Linear Regression
2. Health risk analysis based on AQI
3. Alert system for unsafe air quality
4. Live AQI map visualization

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
- Folium

## How to Run the Project
1. Install required libraries:
   pip install pandas matplotlib scikit-learn folium

2. Run AQI prediction:
   python aqi_prediction.py

3. Run health risk analysis:
   python health_risk_ai.py

4. Run alert system:
   python alert_system.py

5. Run live AQI map:
   python live_aqi_map.py
